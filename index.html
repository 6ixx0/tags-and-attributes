Welcome to this tutorial on html, we would be starting with the different types of tags and attributes.

Anchor Tag

The anchor tag "a" is used to define a hyperlink, it consists of an opening and closing tag, with the content inbetween and it makes use of the attribute "href": <a href="$">link</a>
An example of this is; My instagram profile page

Bold Tag

Used to define bold text; <b>content</b> For example, YOU CAN'T KILL MY VIBE.

Body Tag

Body tag is used to define the documents body;<body>content</body>.

Address Tag

Used to define contact information for the owner of a document; <address>content</address>.

Blockquote Tag

It defines a section that is quoted from another source; <Blockquote>content</Blockquote> ,
For example;

You can't seperate peace from freedom because no one can be at peace unless he has freedom.

Heading Tag

Used to define html headings, ranges from h1-h6, as we move from h1-h6 the size of the heading decreases.

Mark Tag

Used to define a marked text; <mark>content</mark>
For example, TAYLOR.

Image Tag

Used to define an image, represented by img it doesn't have a closing tag and it makes use of the attribute "src" which specifies the path to the image to be displayed and "alt" which gives an additional information about the image incase it doesn't load, it also makes use of the width and height attribute; <img src="$">

Paragraph Tag

It is used to define a pararaph, denoted <p></p>.

Strong Tag

This tag is similar to bold tag;<stong>content</strong>. for example; Run me my MONEY!!!!!.

Cite Tag

This tag defines a citation and displays it in italics; <cite>content</cite>. For example; Tomorrow never dies.

Em Tag

This tag is similar to italics tag; <em>content</em>.

Ordered List Tag

Used to list items accordingly with numbers; <ol>content</ol>.

Unordered List Tag

Used to list items in bullet points; <ul>content</ul>.

Nav Tag

Defines a set of navigation links; <nav>content</nav>

Li Tag

Defines a set of navigation links,is used in ordered lists, unordered lists and in menu lists. <li>content</li>.
