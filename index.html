Welcome to this tutorial on html, we would be starting with the different types of tags and attributes.

Anchor Tag

The anchor tag "a" is used to define a hyperlink, it consists of an opening and closing tag, with the content inbetween and it makes use of the attribute "href": (opening tag)a href="$">link( closing tag)a.
An example of this is; My instagram profile page

Bold Tag

Used to define bold text; (tagname)content(tagname) For example, YOU CAN'T KILL MY VIBE.

Body Tag

Body tag is used to define the documents body;(tagname)content(tagname).

Address Tag

Used to define contact information for the owner of a document.

Blockquote Tag

It defines a section that is quoted from another source; (tagname)content(tagname); ,
For example;

You can't seperate peace from freedom because no one can be at peace unless he has freedom.

Heading Tag

Used to define html headings, ranges from h1-h6, as we move from h1-h6 the size of the heading decreases.

Mark Tag

Used to define a marked text; (tagname)content(tagname)
For example, TAYLOR.

Image Tag

Used to define an image, represented by img it doesn't have a closing tag and it makes use of the attribute "src" which specifies the path to the image to be displayed and "alt" which gives an additional information about the image incase it doesn't load, it also makes use of the width and height attribute.

Paragraph Tag

It is used to define a pararaph, denoted by the letter "p".
